### Overview

This directory contains sample code (c and C++) that demonstrate Cirque's
Gen6 touchpad and CustomMeas solutions. There are three parts to this sample
code: 
	libraries
		-- Arduino libraries for Gen6 Touchpads, CustomMeas, and I2C on the Teensy4.0
	CirqueCustomMeasDemo
		-- Demo app showing how to use CustomMeas device
	CirqueGen6Demo
		-- Demo app showing how to use a touchpad

These projects run on the Cirque 02-000658-00 PCB, Rev E01 or later, with a 
Teensy 4.0 on-board.

Code description and installation instructions are at: 
	www.cirque.com/documentation
